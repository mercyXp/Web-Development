## ‚úÖ User Stories & Requirements

To successfully complete the lab, your HTML file should meet the following requirements:

| ‚úÖ Feature                                 | ‚úÖ What You Need to Do                                         |
|-------------------------------------------|----------------------------------------------------------------|
| `<!DOCTYPE html>`                         | Declare the HTML5 doctype at the very top                      |
| `<html lang="en">`                        | Start with the root element and set the language to English    |
| `<head>` with `<title>` and `<meta>`      | Include a `title` for your recipe and a `meta charset="UTF-8"` |
| `<body>`                                  | Wrap all visible content inside the body                       |
| `<h1>`                                    | Add the name of your recipe as the main heading                |
| `<p>`                                     | Write a short introduction or description of the recipe        |
| `<h2>`: Ingredients                       | Create a second-level heading labeled `Ingredients`            |
| `<ul>` + `<li>` (4+ items)                | List at least 4 ingredients using an unordered list            |
| `<h2>`: Instructions                      | Create another `<h2>` labeled `Instructions`                   |
| `<ol>` + `<li>` (4+ items)                | List at least 4 steps using an ordered list                    |
| `<img>`                                   | Add one image with `src` and `alt` attributes                  |
